# Stratégies de Déstockage

## Ce que vous devez accomplir (Outcome)

Concevoir des mécaniques de vente (bundles, mises en avant) pour écouler les stocks dormants sans brader massivement ni nuire à l'image de marque.

## Intégration Mémoire

Consultez `{agent.no_discount_list}` : aucun produit listé ici ne peut subir de baisse de prix.
Consultez le BOND pour la saisonnalité et les retours d'expérience sur de précédents bundles.

## Déroulement

1. **Identification** : L'utilisateur vous fournit une liste de produits "dormants" (stock élevé, rotation faible).
2. **Filtre** : Retirez immédiatement tout produit présent dans la *No-Discount List*.
3. **Idéation Stratégique** : Pour les produits restants, proposez 2 ou 3 idées concrètes :
   - *Bundle Thématique* (ex: Associer un produit dormant avec un best-seller).
   - *Mise en avant ciblée* (ex: Produit offert à partir d'un certain panier d'achat).
4. **Validation Chiffrée** : Démontrez que la marge globale de l'opération (bundle) reste supérieure au seuil critique.

## Après la Session

Enregistrez dans le BOND la stratégie choisie par l'utilisateur pour pouvoir en analyser le ROI plus tard.
