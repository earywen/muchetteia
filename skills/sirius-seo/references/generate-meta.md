# Générer Meta-balises

## Ce que vous devez accomplir (Outcome)

Fournir un ensemble prêt à l'emploi de balises Title, Meta Description, et attributs Alt d'images pour de nouvelles références produits ou pages, en respectant les limites de caractères des moteurs de recherche (Title < 60 caractères, Meta < 155 caractères).

## Intégration Mémoire

Utilisez les informations de `{agent.editorial_guide_template}` pour conserver le ton de la boutique.

## Déroulement

1. **Identification** : À partir des informations fournies par l'utilisateur (nom du produit, description courte, type d'image), déterminez le mot-clé principal.
2. **Génération Title** : Créez 3 propositions de Title accrocheuses (idéalement entre 50 et 60 caractères).
3. **Génération Meta Description** : Créez 2 propositions de Meta Description incitant au clic, intégrant le mot-clé principal (entre 140 et 155 caractères).
4. **Génération Balise Alt** : Fournissez une description précise et descriptive pour l'image, optimisée pour l'accessibilité et le SEO image.

## Après la Session

Aucune mise à jour mémoire stricte nécessaire, sauf si un nouveau groupe sémantique important a été découvert, auquel cas proposez d'ajouter cela au BOND.
