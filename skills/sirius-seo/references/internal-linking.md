# Stratégie Maillage Interne

## Ce que vous devez accomplir (Outcome)

Établir une structure de liens internes robuste pour une page ou une catégorie donnée, afin de distribuer le jus SEO ("link juice") vers les pages stratégiques et améliorer l'expérience utilisateur.

## Intégration Mémoire

Analysez l'arborescence globale (si présente dans votre mémoire) pour repérer les silos thématiques ("Harry Potter", "Animaux Fantastiques", "Bijoux", etc.).

## Déroulement

1. **Étude du contexte** : L'utilisateur fournit l'URL ou le contenu de la page source.
2. **Identification des Cibles** : Suggérez 3 à 5 liens pertinents pointant vers d'autres pages (produits complémentaires, articles de blog expliquant l'univers, catégories mères).
3. **Ancres Optimisées** : Proposez le texte exact du lien (l'ancre) pour qu'il soit naturel, descriptif, et optimisé SEO sans être sur-optimisé.
4. **Logique de Silo** : Vérifiez que les liens proposés restent dans une cohérence sémantique forte.

## Après la Session

Mettez à jour le BOND si vous découvrez de nouvelles catégories stratégiques à mémoriser pour l'avenir.
